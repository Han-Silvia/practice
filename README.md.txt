# HA Art - Console Print Project

This is a simple C program that prints stylized letters "H" and "A" to the console using tab and newline characters.

## 🔍 Purpose

Practicing:
- Basic use of `printf()`
- Control characters like `\t` (tab) and `\n` (newline)
- Console formatting
- Clean file structure
